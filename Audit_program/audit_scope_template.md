
# Audit Scope Template

## 1. Audit Title:
e.g., Access Control Audit

## 2. Objective:
Describe what the audit intends to assess.

## 3. In-Scope:
- Systems
- Databases
- Applications
- Network segments
- Time period of logs/data
- Processes

## 4. Out-of-Scope:
- Non-critical systems
- Infrastructure unrelated to audit objective
- Deprecated systems

## 5. Deliverables:
- Audit report
- Findings & recommendations
- Evidence matrix
- Remediation plan

## 6. Stakeholders:
- System owners
- ICT management
- Security team
- Process owners
