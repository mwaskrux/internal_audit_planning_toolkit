# IT Audit Universe

The audit universe represents the complete list of auditable components within the organization.

## 1. Infrastructure
- Servers (Windows, Linux)
- Databases (MSSQL, MySQL, PostgreSQL)
- Virtualization platforms (VMware, Hyper-V)
- Storage arrays and SAN
- Network infrastructure (switches, routers, firewalls)

## 2. Security Domains
- Identity & Access Management
- Endpoint protection
- Logging and monitoring (SIEM, Wazuh, Elastic)
- Patch & vulnerability management
- Incident response

## 3. Operations
- Service desk operations
- Change management
- Asset management
- Backup & restoration
- DR & business continuity

## 4. Cloud Environments
- Azure AD
- Microsoft 365 security
- Cloud network configuration
- Cloud identity governance

## 5. Third Parties
- Vendors with system access
- Managed service providers
- Cloud hosting partners

## 6. Business Applications
- ERP
- HR systems
- Revenue cycle systems
- Clinical/Health systems (for healthcare)
